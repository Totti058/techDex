# Tech Dino Dex

Tech Dino Dex es una aplicación frontend que implementa un “Dex” de criaturas Tek de Ark: Survival Evolved, utilizando renderizado dinámico en el DOM a partir de datos estructurados.

## Enfoque

El proyecto prioriza:

- Estructura semántica clara
- Separación de responsabilidades (HTML / CSS / JS)
- Renderizado dinámico basado en estado
- Código legible y mantenible
- Flujo de trabajo con feature branches

## Arquitectura

La información de las criaturas se gestiona como un array de objetos en JavaScript.  
El renderizado se realiza mediante funciones puras que actualizan el DOM según filtros o búsquedas aplicadas por el usuario.

## Stack

HTML5 · CSS3 · JavaScript (Vanilla) · Git · Netlify
